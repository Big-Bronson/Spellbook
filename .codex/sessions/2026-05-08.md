### Lessons and Gotchas
- **OrderedDictionary does not have `.ContainsKey()`**: `[ordered]@{}` uses `.Contains()` instead. This is easy to miss because regular hashtables have both methods, but ordered dictionaries will throw `InvalidOperation` when you call `.ContainsKey()`. Both files that use ordered hashtables in this codebase were burned by this.
- **API keys should live in credential manager, not command lines**: Passwords and tokens in parameter lists leak into PowerShell history. Moving to `Get-StoredSecret` is cleaner and more secure for multi-engineer teams.
- **Publish validation should be opinionated and strict**: The new `Publish.ps1` gates on git state, manifest validity, and changelog entries. This prevents accidental publishes of broken or undocumented versions — the cost is worth it for a PSGallery module.
